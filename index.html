SPRITE SHEET TO GIF

100%

ABOUT THIS TOOL

Turn sprite sheets into animated GIFs without uploading your artwork. Configure the slice, preview every frame, and export directly in the browser.

  • Grid and crop controls
  • Transparent GIF support
  • Zoomable source editor
  • Local browser processing

Built with JavaScript and gif.js. Based on the original collidingScopes project and maintained on GitHub.
